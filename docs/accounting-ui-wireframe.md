# 代理记账公司内部营运系统 UI 草案

该文档提供系统页面结构和关键界面元素，用于指导前端原型设计。

## 顶部导航

- 仪表盘
- 线索
- 产品
- 流程
- 客户与合同
- 工单
- 票据中心
- 会计
- 税务
- 薪资
- 质检
- 报表
- 系统

## 1. 仪表盘

展示经营、交付、财税等核心 KPI。

- 当月续费率、ARPU、在制工单数量。
- 超时工单、税负异常客户提醒。

## 2. 线索

- 线索列表：来源、评分、跟进人、状态。
- 线索详情：跟进记录、商机转化按钮。

## 3. 产品

- 产品目录：按产品线/品类筛选。
- SKU 详情：价格、可选计费方式、关联工单模板。

## 4. 流程

- 模板库：按 SKU/地区/行业展示流程模板。
- 服务追踪器：甘特图、阻塞中心、交接记录。

## 5. 客户与合同

- 客户档案：工商信息、合同列表、账单计划。
- 合同详情：套餐、价格、审批轨迹、收款台账。

## 6. 工单

- 工单池：按状态/优先级分栏。
- 工单详情：里程碑进度、SLA 计时器、暂停记录。

## 7. 票据中心

- 上传清单、缺件催收。
- OCR 识别结果与凭证生成入口。

## 8. 会计

- 凭证流水线视图：制证→初审→复核→记账→月结。
- 科目余额、异常波动预警。

## 9. 税务

- 申报日历：按税种与属地显示节点。
- 申报回执归档，税负异常提示。

## 10. 薪资

- 批次列表：考勤导入、薪资计算、个税申报。
- 员工薪资条下载与签收状态。

## 11. 质检

- 质检任务列表：随机/规则抽查。
- 缺陷率统计与问题复盘。

## 12. 报表

- 经营、交付、财税看板自定义。
- 报表订阅与导出。

## 13. 系统

- 角色与权限管理。
- 配置中心与集成管理。
